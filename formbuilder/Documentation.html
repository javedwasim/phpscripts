Documentatoin -- class: mysqlForm

  • Class : mysqlForm
    1. This class can print a mysql result resource and Create HTML form for MySql table.
    2. Initialization : $obj=new mysqlForm($linkresource,$databasename)
    3. Argument 1 : The link.such as :
      $link=mysql_connect('localhost', 'user', 'password');
    4. Argument 2 : The table name. Suppose we, will work with 'employee' table.
    5. Example: Now this class can be initialized as : $obj=new mysqlForm($link,'employee');
    6. Form fields and there labels are organized in table.
  • Method : printResult($result,$format="border=1")
    1. This method prints a result source as table.
    2. Argument 1 : the result resource. such as : $result
    3. Argument 2 : (Optional) format of table.
    4. Example: $obj->printResult($result,"border=1,cellpadding=2")
    5. Column name used as table header.(see test.php for details)
  • Method : printForm($table,$action,$skip="-1")
    1. This method creats an HTML form for an stracture of MySql table
    2. Argument 1 : Table name, for which the from will be created.
    3. Argument 2 : Action attrebute of form.
    4. Argument 3 : (Optional)Index of columns to skip. For whom form field is not required.
    5. This skip index list should be Comma separated string format. Such as : "0,4,6"
      it skips 0,4 and 6 indexd columns of table and dont create form field for them.
    6. The value of 'method' attrebute of form is POST.
    7. Name of field :All fields of form will be named and labeled as relevent column name of MySql table
    8. Length : Length of a field is :(length of column datatype+5).
      Such as, for varchar(30) column, field size will be 35.
    9. Default field type is text.
    10. If length is more then 50, the field will be text area.
    11. If the column name of table is password, the field will be password.
    12. For ENUM columns, <select> field will be created containing its values.
    13. When submitted, form will submit to 2nd argumented value.
    14. Example : $obj->printForm("user","insert.php",'0,3');
      It creates a form for all fields of 'user' table without 0 and 3 indexed column.
      When submitted, it will submit to 'insert.php' file.
